    Description du projet

    Ce projet a été réalisé dans le cadre d' une opération d'aide aux commerçants touchés par le covid-19 proposée par l'ENI. Le but de cette opération est de réaliser un site WEB pour les commerçants afin de les aider à vendre leurs produits. Nous avons travaillé à 3 personnes sur ce projet.     • permettre l'authentification des commerçants.

    • permettre à l'association de gérer ses infos et son actualités.
    • permettre à l'association et ses commerçants de diffuser leurs actualités auprès des visiteurs du site.
    • permettre au visiter de chercher un commerce par nom ou par catégorie de commerce.

  • Novembre 2019 - Janvier 2020 - Développement d'une application de gestion de magasins

    • Lien Github : https://github.com/fabrice-noel29/Les-bleus

    Description du projet

    Ce projet a été réalisé dans le cadre de ma formation de développeur .NET proposée par IB-Cegos. Nous avons travaillé à 4 personnes sur ce projet. Le but de ce projet est de réaliser une application de gestion de magasin. La thématique du magasin était au choix. Nous avons choisi les vélos. Ce projet était divisé en 3 parties :
    -une application ASP.net dont le rôle est de :

    • permettre l'authentification des vendeurs.
    • permettre au vendeur de consulter le stock global.
    • permettre au vendeur de faire une demande d'ajout de stock pour un type de produit.
    • permettre au vendeur de consulter les commandes effectuées par tout les autres vendeurs.
    • permettre à l'administrateur de créer de nouveau vendeur.
    • permettre à l'administrateur de valider les demandes d'ajout de stock.
    -une application UWP dont le rôle est de :
    • permettre au vendeur de vendre des produits et créer des commandes en étant authentifié.
    • permettre au vendeur d'indiquer au client le prix total de la commande à chaque étape d'ajout ou de retrait de produit.
    -et une application librairie qui regroupe les éléments communs au deux applications ci-dessus.

    Technologies Utilisées
    • Programmation : C#
    • I.D.E : Visual Studio Community
    • SGBD : SQL Server
    • Framework : .NET, ASP.net MVC, UWP, Bootstrap, NewtonSoft.json, Entity Framework
    • Gestion de projet : Trello, Redmine
    • Versioning : Git, github
    Résultats Obtenus
    • Application ASP.net et UWP réalisées
    • Synchronisation entre base de données local et base de données de tout les vendeurs en cours
    • Interdiction de passer une commande si l'application UWP n'est pas connecté en cours de développement
